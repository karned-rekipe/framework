# kcrud

Generic async CRUD framework for Python — domain models, use cases, repositories, logger.

Built on clean architecture principles: domain, application, adapters, infrastructure.

## Installation

```bash
pip install kcrud
```

### Optional dependencies

```bash
pip install "kcrud[mongodb]"
pip install "kcrud[duckdb]"
pip install "kcrud[fastapi]"
pip install "kcrud[mcp]"
pip install "kcrud[all]"
```
